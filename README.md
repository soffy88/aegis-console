# Aegis Console

Next.js dashboard for [Aegis](https://github.com/soffy88/aegis). Runs at :3010.

## Run

```bash
pnpm install
pnpm dev
```

## Configure

Copy `.env.local.example` to `.env.local` and fill in. The console expects
the Aegis backend at `NEXT_PUBLIC_AEGIS_API` (default `http://localhost:8080`).

## Test

```bash
pnpm test
```
